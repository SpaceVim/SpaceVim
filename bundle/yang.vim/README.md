yang.vim
========

YANG syntax highlighting and other niceties for VIM. Use with [Vundle](https://github.com/VundleVim/Vundle.vim) via:

```vim
Plugin 'nathanalderson/yang.vim'
```

Don't forget to run `:PluginInstall`.
