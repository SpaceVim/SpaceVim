## 1 基本语法

### 1.1 变量的命名

Vim 的变量名称可以使用字母、下划线、数字组成，区分大小写，并且不可以以数字开头。

### 1.2 变量的作用域

Vim 脚本中，变量的作用域支持如下类型：

- `g:`：全局作用域
