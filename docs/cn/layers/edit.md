## 模块简介

该模块提升了 SpaceVim 的文本编辑体验，提供更多种文本对象。

## 功能特性

- 修改围绕当前光标的符号
- 重复编辑
- 多光标支持
- 对齐文档内容
- 高亮行为符号
- 自动载入 editorconfig 配置，需要 `+python` 或者 `+python3` 支持

## 模块选项

- `textobj`: specified a list of text opjects to be enabled, the avaliable list is :`indent`, `line`, `entire`
