## 模块描述

SpaceVim ui 模块提供了一个 IDE-like 的界面，包括状态栏、文件树、语法数等等特性。

## 启用模块

可通过在配置文件内加入如下配置来启用该模块：

```toml
[[layers]]
  name = "ui"
```

## 集成插件

- [mhinz/vim-startify](https://github.com/mhinz/vim-startify)
- [majutsushi/tagbar](https://github.com/majutsushi/tagbar)
