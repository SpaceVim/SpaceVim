## 模块简介

该模块主要为 SpaceVim 提供了 swift 开发支持。

## 功能特性

- 语法高亮
- 语法检查

## 启用模块

可通过在配置文件内加入如下配置来启用该模块：

```toml
[[layers]]
  name = "lang#swift"
```

## 快捷键

| 按键      | 功能描述                 |
| ----------- | ----------------------- |
| `SPC l k`   | jumping to placeholders |
