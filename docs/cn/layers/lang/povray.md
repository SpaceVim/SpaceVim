## 模块简介

该模块为 SpaceVim 提供了 [POV-Ray](https://www.povray.org/) 语言的支持，包括语法高亮，图片生成等功能。

## 启用模块

可通过在配置文件内加入如下配置来启用该模块：

```toml
[[layers]]
  name = "lang#povray"
```
