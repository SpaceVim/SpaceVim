## 模块简介

这一模块为 SpaceVim 提供了 [Slim](http://slim-lang.com/) 语言的语法高亮支持。。

## 启用模块

可通过在配置文件内加入如下配置来启用该模块：

```toml
[[layers]]
  name = "lang#slim"
```
