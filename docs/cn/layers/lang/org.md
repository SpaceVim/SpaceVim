## 模块简介

这一模块为 SpaceVim 提供了 org 的编辑支持，包括格式化、自动生成文章目录、代码块等特性。

## 启用模块

可通过在配置文件内加入如下配置来启用该模块：

```toml
[[layers]]
  name = "lang#org"
```
