## 模块简介

这一模块为 SpaceVim 提供了 WebAssembly 的开发支持。采用 [vim-wasm](https://github.com/rhysd/vim-wasm) 这一插件。

## 启用模块

可通过在配置文件内加入如下配置来启用该模块：

```toml
[[layers]]
  name = "lang#WebAssembly"
```
