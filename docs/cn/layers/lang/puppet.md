## 模块简介

该模块为 SpaceVim 提供了 Puppet 语言开发支持。

![puppet ide](https://user-images.githubusercontent.com/13142418/46022200-bb426f00-c114-11e8-84bc-d52dc83806c6.png)

## 功能特性

- 语法高亮

## 启用模块

可通过在配置文件内加入如下配置来启用该模块：

```toml
[[layers]]
  name = "lang#puppet"
```
