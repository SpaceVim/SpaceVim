## 模块简介

这一模块为 Haskell 开发提供支持，包括代码补全、语法检查、代码格式化等特性。

## 启用模块

可通过在配置文件内加入如下配置来启用该模块：

```toml
[[layers]]
  name = "lang#haskell"
```

## 功能特性

- 代码补全
- 语法检查
- 跳转定义处
- 查询函数引用
- lsp 支持
