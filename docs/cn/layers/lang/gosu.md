## 模块简介

这一模块为在 SpaceVim 中进行 gosu 开发提供了支持。

## 启用模块

可通过在配置文件内加入如下配置来启用该模块：

```toml
[[layers]]
  name = "lang#gosu"
```

## 快捷键

在编辑 gosu 文件时，可通过快捷键 `SPC l r` 快速异步运行当前文件，运行结果会展示在一个独立的执行窗口内。
