## 模块简介

这一模块为在 SpaceVim 中进行 FSharp 开发提供了支持.

## 功能特性

- 语法高亮、对齐，由插件 [vim-fsharp](https://github.com/wsdjeg/vim-fsharp) 提供。

## 启用模块

可通过在配置文件内加入如下配置来启用该模块：

```toml
[[layers]]
  name = "lang#fsharp"
```
