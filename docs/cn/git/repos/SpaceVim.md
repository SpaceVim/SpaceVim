# [开发者文档](../) > 源码仓库 

SpaceVim 的源码托管于在线仓库，可以克隆至本地查看源码。

```
git clone https://spacevim.org/git/repos/SpaceVim/
```

由于代码早期托管于 GitHub，因此，对于部分老用户，需要手动修改远程仓库地址：

```
cd ~/.SpaceVim
git remote set-url origin https://spacevim.org/git/repos/SpaceVim/
```
