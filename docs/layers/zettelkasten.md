## Description

This layer adds support for zettelkasten in neovim.

## Install

To use this configuration layer, update your custom configuration file with:

```toml
[[layers]]
  name = "zettelkasten"
```

## Layer options

- `zettel_dir`: set the default zettelkasten directory
