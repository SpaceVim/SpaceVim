## Description

This layer is for latex development. [vimtex](https://github.com/lervag/vimtex) is incuded in this layer.

## Features

- Code completion
- Syntax highlighting
- Syntax lint

## Install

### Layer

To use this configuration layer, update custom configuration file with:

```toml
[[layers]]
  name = "lang#latex"
```
