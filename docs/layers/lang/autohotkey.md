## Description

This layer adds [AutohotKey](https://www.autohotkey.com/) language support to SpaceVim.

## Features

- syntax highlighting
- code completion

## Install

To use this configuration layer, update your custom configuration file with:

```toml
[[layers]]
  name = "lang#autohotkey"
```
