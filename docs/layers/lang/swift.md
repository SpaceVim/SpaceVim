## Description

This layer is for swift development in SpaceVim.

## Install

To use this configuration layer, update custom configuration file with:

```toml
[[layers]]
  name = "lang#swift"
```

## Features

- auto-completion
- syntax checking
- goto definition
- refernce finder
- language server protocol (need lsp layer)
