## Description

`lang#cmake` layer provides syntax highlighting for cmake script.

## Install

This layer is not enabled by default.
To use this configuration layer, update your custom configuration file with:

```toml
[[layers]]
  name = "lang#cmake"
```
## Features

- syntax highlighting
