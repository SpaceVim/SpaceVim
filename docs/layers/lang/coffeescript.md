## Description

This layer is for CoffeeScript development, and it includes vim-coffeescript.

## Features

- Code completion
- Syntax highlighting
- Indent
- Code formatting
- REPL support

## Install

To use this configuration layer, update custom configuration file with:

```toml
[[layers]]
  name = "lang#coffeescript"
```
