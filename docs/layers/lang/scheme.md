## Description

This layer adds Scheme language support to SpaceVim. Scheme is known as mit-scheme, and the website is: <http://www.scheme-reports.org>


## Features

- code runner
- REPL support

## Install

To use this configuration layer, update custom configuration file with:

```toml
[[layers]]
  name = "lang#scheme"
```
