## Description

This layer is for clojure development, and it includes vim-clojure-static and vim-fireplace.

## Features

- Code completion
- Syntax highlighting
- Indent
- Code formatting
- REPL support

## Install

### Layer

To use this configuration layer, update custom configuration file with:

```toml
[[layers]]
  name = "lang#clojure"
```
