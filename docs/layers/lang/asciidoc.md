## 模块简介

该模块为 SpaceVim 提供了 asciidoc 文件夹编辑支持。该模块包括插件：

- `wsdjeg/vim-asciidoc`
- `dahu/VimRegStyle`
- `dahu/vimple`
- `dahu/Asif`

## 启用模块

可通过在配置文件内加入如下配置来启用该模块：

```toml
[[layers]]
  name = "lang#asciidoc"
```
