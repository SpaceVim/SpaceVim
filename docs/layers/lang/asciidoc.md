## Description

This layer is for editing AsciiDoc file. Following plugins are included in this layer:

- `wsdjeg/vim-asciidoc`
- `Raimondi/VimRegStyle`

## Install

To use this configuration layer, update custom configuration file with:

```toml
[[layers]]
  name = "lang#asciidoc"
```
