## Description

This layer is for haskell development.

## Install

To use this configuration layer, update custom configuration file with:

```toml
[[layers]]
  name = "lang#haskell"
```

## Features

- auto-completion
- syntax checking
- goto definition
- refernce finder
- language server protocol (need lsp layer)
