## Description

This layer is for haskell development.

## Layer Installation

To use this configuration layer, add `call SpaceVim#layers#load('lang#haskell')` to your custom configuration file.

## Features

- auto-completion
- syntax checking
- goto definition
- refernce finder
- language server protocol (need lsp layer)
